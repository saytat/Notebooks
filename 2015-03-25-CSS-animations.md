## CSS Animations

1. select the element to be animated: 
        html_element {
     			animation-name: flip;
                animation-duration: 2s;
                animation-timing-function: ease;
        }

2. define the animation:
		@keyframes flip {
        		start {
                }
                
                stop {
                }
        }
   OR
       
       @keyframes flip {
                    0% {
                    }
                    
                    33% {
                    }
                    
                    66% {
                    }
                    
                    100% {
                    }
            }
